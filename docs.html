Authentication

All requests require your API key in the header:

x-api-key: your_api_key_here

Example with headers

curl -H "x-api-key: ua_abc123..." https://api-keys-5tdu.onrender.com/ai?prompt=hello

Securing Your Key

  • Never commit your key to version control
  • Use environment variables: API_KEY=process.env.UNIFIED_API_KEY
  • Rotate your key if exposed
  • Each user should have their own key

Environment Variable Setup

# .env file UNIFIED_API_KEY=ua_your_key_here # In your code api_key = process.env.UNIFIED_API_KEY

Error Handling

API errors return JSON with status codes and error messages.

401 - Unauthorized

{ "detail": "API Key missing. Please provide 'x-api-key' in the header." }

403 - Forbidden

{ "detail": "Invalid API Key. Please check your key or upgrade your plan." }

429 - Rate Limited

{ "detail": "Rate limit exceeded. You have 100/100 calls this month. Upgrade to Pro." }

502 - Provider Down

{ "detail": "API provider is currently down. Try again in a moment." }

Best Practices

  • Always check the status field
  • Implement retry logic for 502/503 errors
  • Handle 429 rate limits gracefully
  • Log errors for debugging

Rate Limiting

UnifiedAPI enforces rate limits based on your plan:

Free Tier
100 API calls per month
Pro Tier (R150/month)
10,000 API calls per month

How It Works

  • Each API call counts as 1 call (regardless of endpoint)
  • Resets on the first day of each month
  • Exceeding your limit returns 429 error
  • You can see your usage in your dashboard
💡 Pro Tip
Cache responses when possible to reduce API calls. Check your usage regularly in the dashboard to avoid hitting your limit.

const API_KEY = process.env.UNIFIED_API_KEY; const BASE_URL = "https://api-keys-5tdu.onrender.com"; async function callAPI(endpoint, params = {}) { try { const query = new URLSearchParams(params); const url = `${BASE_URL}${endpoint}?${query}`; const response = await fetch(url, { headers: { "x-api-key": API_KEY } }); if (!response.ok) { throw new Error(`API Error: ${response.status}`); } return await response.json(); } catch (error) { console.error("API call failed:", error); throw error; } } // Usage const joke = await callAPI("/joke"); console.log(joke.data);
